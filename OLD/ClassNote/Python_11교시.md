# 앙상블 기법

앙상블 학습을 통한 분류는 여러 개의 분류기를 생성하고 그 예측을 결합함으로써 보다 정확한 최종 예측을 도출하는 기법

1. 보팅  _병렬 (알고리즘이 모두 다르면)

2. 배깅  _병렬 (알고리즘이 모두 같으면___랜덤 포레스트 )

   다양하게 하기 위해 부트스트랩(복원샘플링 사용)

   변수도 다양하게

3. 부스팅 _직렬 (아다부스트, XG부스트(대회 1등)<-오차가 난 데이터에 가중치를 줌, 에이다(아다)부스트

4. 스태킹
